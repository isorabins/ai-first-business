# AI-First Business

This repo documents the build of an AI-first small business — real workflows, real agent skills, and real lessons from building in production.

Follow the journey on YouTube: **[Life with AI with Iso](https://www.youtube.com/@isorabins)**

---

## What's in here

Skills, prompts, and workflow files from building **Noko** — an AI agent that runs the operational layer of [Forage Bali](https://foragebali.com), a wild food foraging experience in Bali, Indonesia.

These aren't theoretical. They were built to solve real problems and are running in production.

---

## Structure

```
/skills/              # Reusable agent skill files
/workflows/           # Step-by-step workflow docs for specific tasks
/prompts/             # Prompt templates that work
```

Each folder has its own README explaining what's inside and how to use it.

## How to use this

These files are designed for AI agents (Claude, GPT, etc.) but the concepts apply to any LLM. Drop them into your agent's context, adapt the API endpoints and credentials to your stack, and use them as a starting point.

If something doesn't work for your setup, open an issue — happy to help.
