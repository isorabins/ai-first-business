# Skills

Reusable skill files for AI agents — drop these into your agent's context to give it a specific capability.

## What's a skill?

A skill is a markdown file your agent reads before tackling a specific type of task. It gives the agent the context, rules, and step-by-step instructions it needs to do the job reliably.

Think of it like a standard operating procedure — but for an AI.

## How to use

1. Drop the skill file into your agent's workspace
2. Reference it in your agent's `TOOLS.md` or `SOUL.md` so it knows the skill exists
3. When the agent needs to run that workflow, it reads the skill file first
